## Login

* URL
    - ```/login```
* Method
    - POST
* Request Body
    - ``email`` 
    - ``password`` 
* Response
```json
{
    "success": true,
    "user": {
        "id": 9,
        "name": "Wildan",
        "email": "email@email.com",
        "email_verified_at": null,
        "no_hp": "no hape",
        "role": "admin",
        "created_at": "2023-06-15T12:48:56.000000Z",
        "updated_at": "2023-06-15T13:11:14.000000Z",
        "seminar_applied": "[2, 1]"
    },
    "token": "$Token"
}
```

## Add Seminar
* URL
    - ```/seminars```

    - POST
* Headers
    - ``Authorization`` : ``Bearer <token>``
* Request Body
    - ``name`` 
    - ``short_description``
    - ``full_description``
    - ``speaker``
    - ``quota``
    - ``date_and_time`` (YYYY-MM-DD HH-MM-SS)
    - ``lokasi`` (Offline/Online)
    - ``alamat``
    - ``category``

```json

    "message": "Seminar Berhasil ditambahkan"
}
```
